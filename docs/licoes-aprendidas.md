# 📚 Lições Aprendidas — Sistema IoT de Monitoramento Ambiental com ESP32

## 📌 Visão Geral

Este documento apresenta as principais lições aprendidas durante o desenvolvimento do Sistema IoT de Monitoramento Ambiental com ESP32.

O projeto permitiu aplicar na prática conceitos de Internet das Coisas, sistemas embarcados, sensores, atuadores, comunicação Wi-Fi, computação em nuvem e automação.

---

## 🧠 Aprendizado sobre ESP32

Durante o desenvolvimento do projeto, foi possível compreender melhor o funcionamento do ESP32 como microcontrolador aplicado a soluções IoT.

Principais aprendizados:

* Utilização do ESP32 como unidade central de processamento
* Configuração de pinos digitais e analógicos
* Leitura de sensores conectados ao microcontrolador
* Controle de atuadores
* Uso da comunicação serial para testes
* Comunicação Wi-Fi integrada
* Envio de dados para plataformas externas

---

## 🌐 Aprendizado sobre Internet das Coisas

O projeto demonstrou como dispositivos físicos podem ser conectados à internet para coletar, processar e transmitir dados.

Principais aprendizados:

* Conceito de dispositivos conectados
* Coleta de dados em tempo real
* Envio de informações para a nuvem
* Monitoramento remoto
* Integração entre hardware e software
* Uso de plataformas IoT para visualização de dados

---

## 🔧 Aprendizado sobre Sensores

O projeto possibilitou compreender como sensores podem ser utilizados para capturar informações do ambiente.

Principais aprendizados:

* Leitura de temperatura
* Leitura de umidade do ar
* Leitura de luminosidade
* Leitura de umidade do solo
* Simulação de qualidade do ar
* Uso de potenciômetro para representar variável analógica
* Interpretação dos valores coletados

---

## 🚨 Aprendizado sobre Atuadores

Além da coleta de dados, o projeto mostrou como o sistema pode reagir automaticamente a determinadas condições ambientais.

Principais aprendizados:

* Acionamento de LED
* Acionamento de buzzer
* Uso de relé para representar controle de dispositivos externos
* Criação de alertas automáticos
* Resposta do sistema a condições fora dos limites definidos

---

## ☁️ Aprendizado sobre ThingSpeak

A integração com o ThingSpeak permitiu compreender como dados coletados por sensores podem ser enviados para uma plataforma em nuvem.

Principais aprendizados:

* Criação de canal de monitoramento
* Envio de dados por requisições HTTP
* Organização das variáveis em campos
* Visualização por gráficos
* Monitoramento remoto
* Armazenamento histórico de medições

---

## 🧪 Aprendizado sobre Simulação no Wokwi

O uso do Wokwi foi importante para testar o projeto antes de uma montagem física.

Principais aprendizados:

* Simulação do ESP32
* Teste de sensores e atuadores
* Validação da lógica do código
* Visualização do circuito
* Uso do monitor serial
* Simulação de diferentes condições ambientais

---

## 🧱 Aprendizado sobre Modelagem no Tinkercad

A modelagem no Tinkercad contribuiu para visualizar a possível estrutura física do protótipo.

Principais aprendizados:

* Organização dos componentes no gabinete
* Representação física do sistema
* Planejamento da disposição dos sensores
* Visualização da montagem
* Aproximação do projeto a uma aplicação real

---

## 💻 Aprendizado sobre Programação em C++

O desenvolvimento do código permitiu aplicar conceitos de programação voltados a sistemas embarcados.

Principais aprendizados:

* Uso de funções
* Configuração de constantes
* Leitura de sensores
* Controle de atuadores
* Estruturas condicionais

## 🔐 Aprendizado sobre Segurança

O projeto também permitiu refletir sobre boas práticas de segurança em soluções conectadas.

Principais aprendizados:

* Proteção de chaves de API

* Evitar publicação de credenciais reais

## ✅ Conclusão

O desenvolvimento deste projeto contribuiu para consolidar conhecimentos práticos em IoT, ESP32, sensores, atuadores, computação em nuvem e automação.

A experiência demonstrou como é possível criar uma solução funcional de monitoramento ambiental utilizando tecnologias acessíveis e aplicáveis a diferentes cenários, como casas inteligentes, agricultura, laboratórios, cidades inteligentes e ambientes industriais.
